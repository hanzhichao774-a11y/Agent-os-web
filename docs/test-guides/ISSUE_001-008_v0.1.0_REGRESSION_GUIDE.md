## 场景一：LLM 动态生成 Skill（Issue 001 核心验证）

### 1A — 生成一个自定义 Skill

**用户操作**：在对话框中输入：

```
帮我生成一个 Skill，能统计知识库中每个文档的字数并排序
```

**预期后端日志**：
```
[SKILL_GEN] 已写入 Skill 文件: /path/to/skills/word_count.py
```
（文件名由 LLM 决定，可能不同）

**预期响应**：
```
已成功生成并注册技能「文档字数统计」

- 文件：`skills/word_count.py`
- 类别：analysis
- 说明：统计知识库中每个文档的字数并按升序/降序排列
- 后续执行相关操作时，系统将自动通过此技能执行
```

**验证文件**：
```bash
cat backend/skills/word_count.py
# 应包含 SKILL_META 和 def run( 函数
# 不应包含硬编码关键词列表或正则
python -c "import ast; ast.parse(open('backend/skills/word_count.py').read()); print('语法校验通过')"
```

### 1B — 生成无效描述时的错误处理（如网络正常但描述过于模糊）

**用户操作**：

```
帮我创建一个技能
```

**预期**：LLM 应尝试生成一个通用功能的 Skill，或返回包含"技能"关键词的成功消息。如果网络异常导致 LLM 调用失败，应看到：

```
技能生成失败：LLM 调用异常（...）。请稍后重试。
```
（不会有文件写入 skills/ 目录）

## 场景二：entity_exclusion 使用 LLM 解析意图（Issue 008 核心验证）

### 前提

- 需要先完成至少一次实体抽取，使知识库中有实体数据

### 2A — 排除文档实体（LLM 解析文件名和操作方向）

**用户操作**：

```
把 test_company.md 里的实体都排除掉
```

**预期后端日志**：
```
[EXCLUSION_INTENT] ...
```
（新代码路径的日志）

**预期响应**：
```
已将来源为「test_company.md」的 **XX** 个实体标记为排除。

图谱将不再显示这些实体。...
```

### 2B — 非标准表达测试（不使用预设关键词）

**用户操作**（故意不用"排除"这个词）：

```
把 report.pdf 的那些节点从图谱里去掉
```

**预期**：LLM 应理解"去掉"= exclude，成功执行排除操作（旧实现因为没有"去掉"这个关键词会返回"请指定文档名称"的错误提示）

### 2C — 恢复操作

**用户操作**：

```
把刚才排除的 test_company.md 的实体恢复回来
```

**预期**：LLM 解析为 restore 操作，恢复实体并返回恢复数量。

### 2D — 无文件名时的引导

**用户操作**：

```
帮我排除一些实体
```

**预期**：
```
请指定要排除的文档名称。

当前实体来源：xxx.md, yyy.pdf

示例：「排除 xxx.md 的实体」
```

## Before / After 对比

### Skill 创建路径

| | 旧实现（Issue 001 修复前） | 新实现（Wave 3 修复后） |
|---|---|---|
| 触发方式 | 固定关键词（"实体抽取"、"排除实体"） | LLM 理解任意自然语言描述 |
| Skill 内容 | 复制静态字符串模板 | LLM 动态生成 Python 代码 |
| 代码校验 | 无 | `ast.parse` 语法 + `SKILL_META`/`run` 结构校验 |
| 失败处理 | 可能写出语法错误的文件 | 拒绝写文件并返回可读错误 |
| 可定制性 | 完全相同（所有项目使用同一模板） | 每次生成可按用户需求定制 |

### entity_exclusion 意图解析

| | 旧实现（Issue 008 修复前） | 新实现（Wave 3 修复后） |
|---|---|---|
| 操作方向判断 | `is_restore = any(kw in msg for kw in 5 固定词)` | LLM 解析自然语言 |
| 文件名提取 | 3 条正则（文件扩展名匹配） | LLM 从上下文中提取 |
| 错误处理 | 表达稍有不同即报"请指定文档名称" | LLM 能理解同义表达 |
| 日志追踪 | 无 | `[EXCLUSION_INTENT]` 打印 intent 结果 |
